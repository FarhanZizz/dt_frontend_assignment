# Deep Thought - Task Assignment

A frontend application for managing and displaying task assignments with a modern UI featuring a navbar, notice board, asset grid, and journey tracking.

## Project Structure

```
dt_frontend_assignment/
├── index.html
├── css/
│   ├── index.css
│   ├── layout.css
│   └── components.css
├── js/
│   ├── data.js
│   └── main.js
└── assets/
    ├── logo.svg
    ├── home-icon.svg
    ├── settings-icon.svg
    ├── notifications-icon.svg
    ├── profile-icon.svg
    ├── more-icon.svg
    ├── question-mark.svg
    ├── meeting.svg
    └── schedule.svg
```

## Features

- **Navigation Bar** - Logo and action buttons (Home, Settings, Notifications, Profile, More)
- **Notice Board** - Animated notice panel with close functionality
- **Project & Task Display** - Dynamic content rendering for project titles and task descriptions
- **Asset Grid** - Responsive grid for displaying asset cards
- **Floating Action Buttons** - Quick access buttons for help, meetings, and scheduling
- **Journey Board** - Collapsible sidebar for tracking task progress

## Getting Started

1. Open `index.html` in a web browser
2. The page loads data from `js/data.js`
3. Dynamic content is rendered via `js/main.js`

## Styling

- **index.css** - Global styles and typography
- **layout.css** - Layout and structural styles
- **components.css** - Component-specific styles
